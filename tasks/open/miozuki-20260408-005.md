## Tooling to set up

- `scripts/gen-image.mjs` — Node 20+ ESM, no deps. CLI flags: `--prompt`, `--out`, `--aspect` (default 16:9), `--size` (default 2K), `--model` (default `gemini-3.1-flash-image-preview`), `--ref` (repeatable).
- `scripts/prompts/_templates.md` — brand-locked starter scaffolds (editorial mood band, blog hero, atmospheric background). Each scaffold deliberately incomplete to force re-reading the research file before each generation.
- `scripts/prompts/<slot>.prompt.md` — sidecar audit-trail file per generation (final expanded prompt + model + date + sign-off-gate decision).
- `public/generated/.gitkeep` — outputs land here, committed as static assets.
- `package.json` — add `"gen-image": "node --env-file=.env.local scripts/gen-image.mjs"`.
