Four positioning decisions that must be written and locked in before any content is published. No code required -- pure copywriting.

**1. Heirloom ethics narrative**
Write the primary brand narrative for the About page and homepage hero. The frame: jewellery designed to be passed down, not replaced. Not "moissanite instead of a diamond" -- the stone that earns its place in the family without the extraction story.

Key copy points to work from:
- Moissanite: 9.25 Mohs hardness, second only to diamond. Built to last generations.
- Pearl: coastal heritage. Natural, timeless.
- Ethical sourcing becomes more meaningful over time, not less.
- "Jewellery for the family, not the occasion."

**2. Dual-category positioning**
Every brand touchpoint should explicitly acknowledge the moissanite and pearl combination. No NZ competitor combines both categories. This is a structural differentiator and a PR angle.

Write: one paragraph for the About page that names both categories and explains why the combination is intentional, not incidental.

**3. NZ provenance narrative**
"NZ-owned" is generic. Write a specific story about what NZ means to Miozuki's aesthetic. One clear paragraph: what landscape, what coast, what cultural or sensory reference is the brand rooted in. This is the journalist hook for Viva and Stuff pitches.

**4. Gifting collection copy**
Write the collection description and framing for the $300-$1,200 gifting tier (pearl studs, moissanite pendants, stacking rings). Frame as curated fine gifting, not entry-level. The copy should make clear this tier is intentional -- a considered gift, not a budget compromise.

All copy is first draft. Requires Ryo review and edit before going live. These are the words the brand will use everywhere; do not rush the review.
