Non-negotiable technical SEO foundation for a headless Next.js + Shopify build. Must be complete before launch.

**Rendering (verify first)**
- Confirm all product and collection pages are SSG (static generation with ISR revalidation) or SSR. Client-rendered pages will not index reliably. This is the most critical item to verify first.

**Structured data (JSON-LD)**
- `Product` schema on every product page: name, image, description, brand, offers (price, availability, currency, URL). `aggregateRating` once reviews exist. Missing `brand` reduces AI-assisted search discovery.
- `FAQPage` schema on: "Moissanite vs Diamond NZ" article and moissanite FAQ page
- `BreadcrumbList` on all pages
- `Organization` schema on homepage

**Canonical tags**
- Block Shopify's default myshopify.com URLs from indexing. Set canonical to miozuki.co.nz on all pages.
- Collection filter variants (colour, size, metal): canonical points to unfiltered collection URL.

**XML sitemap and robots.txt**
- Auto-generate sitemap via Next.js sitemap package
- Submit to Google Search Console on launch day
- robots.txt: block /admin, /cart, /checkout, internal API routes

**Core Web Vitals**
- Use `next/image` with `priority` flag on all above-the-fold product images
- WebP format, correct `sizes` attribute
- Target LCP under 2.5 seconds
- No layout shift from late-loading fonts or images without defined dimensions

**Metadata**
- Unique `title` and `meta description` on every page
- Product pages: `[Product Name] | Miozuki NZ`
- Collections: `[Category] NZ | Miozuki`
- No duplicate titles across collection variants

**Open Graph meta tags (product pages)**
- Add `og:title`, `og:description`, `og:image`, `og:price:amount`, `og:price:currency`, `og:availability` to Next.js `<Head>` on every product page, populated from Shopify Storefront API data
- Required for correct link previews on all platforms (WhatsApp, social, iMessage)

**Internal linking**
- Every blog/education article links to at least one relevant product collection
- Every product collection links to relevant educational content
- Build into article template from the start; not a retroactive fix
