The page is already live and ranking. The featured snippet for "moissanite vs diamond NZ" is currently uncontested. This is the highest-leverage SEO action available with almost no new content needed.

**Changes required:**

1. Direct answer in the first 100 words
   - The opening paragraph must directly answer "is moissanite a good alternative to diamond?" before any other content. Google pulls the featured snippet from the first clear, direct answer on the page.

2. Comparison table
   - Add a table with clear column headers: Cost per carat, Hardness (Mohs), Fire (colour dispersion), Brilliance, Ethical sourcing, Availability in NZ
   - Table should appear high on the page, ideally within the first 300 words

3. FAQ section (6-8 questions)
   - Add a dedicated FAQ section at the bottom of the article
   - Target specific question-format queries: "Is moissanite better than diamond NZ?", "Does moissanite look fake?", "How long does moissanite last?", "Is moissanite ethical?", "Can you tell moissanite from diamond NZ?"

4. FAQPage JSON-LD schema
   - Add FAQPage structured data markup to the page covering the FAQ section
   - This is a developer task; can be done alongside the Phase 1 technical SEO baseline

**Why this is priority:** The snippet is available now. Competitors have comparison content but none have optimised for the snippet format. A well-structured page takes position 1-3 and captures qualified traffic with no new content creation required.

Effort: 2-3 hours (copywriting + dev for schema)
