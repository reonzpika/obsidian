## Goal

Build a fully dynamic blog on the Next.js frontend, pulling content from Shopify via the Storefront API. 23 posts exist on `/blogs/news`. New posts Ting publishes in Shopify admin must appear automatically without any deploy.

## Scope

### Routes to build

- `/blogs/news` — blog listing page
- `/blogs/news/[handle]` — individual article page

### Listing page (`/blogs/news`)

- Query: `blog(handle: "news") { articles(first: 50) { edges { node { handle title publishedAt excerpt image { url altText } tags } } } }`
- Show article card: featured image, title, excerpt, date, tags
- Tag filter links (these exist on the live site and are SEO-indexed): `affordable-engagement-ring`, `bridal-jewellery-nz`, `halo-ring`, `moissanite`, `moissanite-diamond`, `moissanite-earrings`, `moissanite-engagement-ring`, `moissanite-nz`, `moissanite-ring-engagement`, `moissanite-ring-nz`, `moissanite-rings-nz`, `moissanite-studs`, `moissanite-vs-diamond`, `pearl-earrings-nz`, `pearl-earrings-silver`, `pearl-studs`
- ISR: `revalidate: 3600` (1-hour cache — new posts appear within 1 hour, no deploy needed)

### Article page (`/blogs/news/[handle]`)

- Query: `blog(handle: "news") { articleByHandle(handle: $handle) { title publishedAt contentHtml image { url altText } tags author { name } } }`
- Render `contentHtml` as rich HTML (Shopify stores article body as HTML)
- `generateStaticParams`: pre-build all 23 existing handles at deploy time
- Fallback: `blocking` — new posts render on first request, then cache (no deploy needed for new articles)
- ISR: `revalidate: 86400`

### SEO

- `<title>` and `<meta description>` from article title + excerpt
- OG image from article featured image
- Canonical URL: `https://miozuki.co.nz/blogs/news/[handle]`

## Notes

- Shopify Storefront API fully supports blog/article queries — no additional API scopes needed
- Tag filter: `/blogs/news?tag=moissanite-nz` — client-side filter on the fetched article list is fine for 23 posts; no separate API call needed per tag
- Article body comes back as `contentHtml` (HTML string) — render with `dangerouslySetInnerHTML` inside a styled prose container (Tailwind `prose` class)
- Brand styling: Playfair serif for article titles, cream `#F5F0E9` background, burgundy `#7B1E22` accents
