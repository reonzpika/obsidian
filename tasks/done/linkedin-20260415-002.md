```cmd
schtasks /create /tn LinkedInAuthPreflight ^
  /tr "C:\path\to\python.exe C:\Users\reonz\cursor\linkedin\scripts\auth_preflight.py" ^
```

The scheduled Tue/Wed/Thu posts run via `schtasks` 20 min before the slot. If the session has expired (likely every ~2 weeks given current Chrome UA fingerprint), the post fails silently. The pre-flight gives a 24-hour warning window so it can be fixed before the post day.
