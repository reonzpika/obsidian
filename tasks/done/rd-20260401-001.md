Both are existing KiwiSaver members. Cannot opt out — must apply for savings suspension via myIR. Do this promptly after PAYE is set up.

## Steps (each person does this independently via their own myIR)

1. Log in at myir.ird.govt.nz
2. Go to KiwiSaver → Apply for a savings suspension
3. Must have been a KiwiSaver member for 12+ months to be eligible
4. Suspension lasts 3 months to 1 year (can be extended)
5. Submit — takes effect from the following pay period after IRD notifies the employer
