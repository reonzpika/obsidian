Configure Claude Code hooks for automatic memory injection and session management.

Currently: no hooks configured. Memory relies on Claude remembering to read files.

Three hooks to implement (per ai-memory-systems-report and perfect-memory-system-report):

- SessionStart: fires at session start and for every sub-agent. Injects memory index into context automatically. Removes reliance on Claude remembering.
- PreCompact: fires before context compaction. Re-injects critical context (identity + current priorities) to prevent loss during long sessions.
- SessionEnd: writes updated context back to memory files.

Configure in settings.json. Depends on memory structure being defined (fo-20260427-002) before hooks can point to the right files.
