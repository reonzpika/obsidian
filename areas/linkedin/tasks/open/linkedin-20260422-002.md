Autonomous execute pipeline (`scripts/execute_scheduled.py`) requires a Telegram bot to send GH comments for approval and receive replies. Pipeline will not fire for live posts until this is complete.

## Blocker

Cannot log into Telegram to set up the bot.

## Steps

1. Log into Telegram (resolve login issue first)
2. Message @BotFather: run `/newbot`, follow prompts, copy the bot token
3. Message @userinfobot to get your chat ID
4. Add to `C:/Users/reonz/cursor/linkedin/.env`:
   - `TELEGRAM_BOT_TOKEN=<token>`
   - `TELEGRAM_CHAT_ID=<chat_id>`
5. Smoke test: run `python -c "from dotenv import load_dotenv; load_dotenv('.env'); from tools.telegram import send_message; print(send_message('test'))"` from the repo root
6. Run first live post with the autonomous pipeline
