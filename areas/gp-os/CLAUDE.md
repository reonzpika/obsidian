## Rules for clinical/regulatory content

Always require source data before drafting medico-legal content. Verbal descriptions are insufficient.
- Audit findings: need the actual recording sheet or data
- CPD evidence: need the activity record
- Fellowship assessments: need the assessor's original feedback

Read `context/fellowship-assessment-guide.md` before any fellowship-related work.
